# Simple Interest Calculator

A simple bash script that calculates simple interest based on principal, annual rate of interest and time period in years.

## Purpose
The purpose of this project is to provide a quick and easy way to calculate interest for loans or investments using the command line.

## Formula
The script uses the standard formula for simple interest:
**Simple Interest = (P * R * T) / 100**

Where:
* **P** = Principal amount
* **R** = Annual rate of interest
* **T** = Time period in years

## How to run
1. Ensure you have bash installed on your system.
2. Clone this repository.
3. Run the script:
   ```bash
   bash simple-interest.sh
